
A dump of wallpapers I've collected over time.

If you find something you like, great.
If you've got a collection worth sharing, even better.
Feel free to contribute.

---

## Contributing

Pull requests are welcome.

If you're adding wallpapers, please follow a few simple rules:

- Add them to an existing folder if they match its style.
- If they don't fit anywhere, create a new folder with a meaningful name that represents the aesthetic rather than something generic.
Try to keep each folder focused around a single style or theme.

---

## Image Quality

Please avoid low-resolution or heavily compressed images.

The idea is to keep this repository useful on modern high-resolution displays, so whenever possible, contribute the highest quality version you have.

If an image exists in multiple resolutions, prefer the larger one.

There isn't a strict structure beyond keeping similar wallpapers together.

---

## Before Opening a PR

- Avoid duplicates.
- Avoid obvious AI upscales.
- Avoid images with large watermarks.
- Keep folder names clean and descriptive.

---

## Credits

All wallpapers belong to their respective artists and creators.

If you're the creator of an image hosted here and would like it removed, feel free to open an issue and I'll take care of it.

---

Enjoy browsing.

And if you've built up your own wallpaper collection over the years, don't let it rot on a hard drive. Open a pull request and add it here.
